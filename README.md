# Alma de Granja

Landing ecommerce premium construida con React + Vite + Tailwind CSS para mostrar productos del campo, armar un carrito y enviar el pedido completo por WhatsApp.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Variables de entorno

Copia `.env.example` a `.env` y define:

- `VITE_WHATSAPP_NUMBER`
- `VITE_CLOUDINARY_CLOUD_NAME`
- `VITE_CLOUDINARY_API_KEY`
- `VITE_CLOUDINARY_UPLOAD_PRESET`

## Despliegue en Vercel

El proyecto queda listo para desplegar como SPA en Vercel con `vercel.json` y sin dependencias nativas ni binarios externos.
